# Electronic Tasbeeh Counter System

A desktop application designed to help users maintain and track their daily Tasbee7 through organized data logging.

## Features
- **Persistent Storage:** Saves all entries to external files so progress is never lost.
- **Data Analytics:** Automatically calculates daily totals and identifies the most frequent zikr.
- **Progress Tracking:** Review history over days or weeks to maintain consistency.

## Tech Stack
- **Language:** Python
- **Storage:** File I/O (txt/csv)

## Installation & Usage
1. Clone the repository:
   `git clone https://github.com/abdelrahman2helal/Tasbee7a.git`
2. Compile the source code:
   `tasbeeh_console.py -o tasbeeh`
3. Run the application:
   `./Tasbee7a`
